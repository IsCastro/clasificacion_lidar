Generación del sombreado del relieve


La renderización del sombreado del relieve (hillshade), se realiza aplicando la metodología propuesta por Collette Gantenbein (Universidad de Idaho) y publicada por el U.S. Geological Survey bajo el nombre “Creating Shaded Relief for Geologic Mapping using Multiple Light Sources”, sobre la base de un modelo digital de elevaciones calculado para un paso de malla de 1x1 metros.

Este procesado utiliza múltiples focos de iluminación, con objeto de lograr una mayor plasticidad visual, lo que facilita la interpretación de las características geomorfológicas del territorio, reduciendo los contrastes y las zonas excesivamente brillantes o en sombra absoluta.

Tras múltiples pruebas, se han seleccionado las siguientes tres fuentes de iluminación:

Acimut Altitud Transparencia
350º 70º 65%
15º 60º 50%
270º 270º 70%

Todas ellas se combinan mediante transparencias, generando finalmente un modelo de sombreado compuesto, donde cada punto del terreno muestra un valor de brillo.

El procesado de las imágenes para generar el modelo de sombreado se ha implementado mediante un proceso automático en FME.
